# Setup

Just run this in you terminal: 
```bash
./mac-setup-script.sh
```

This script will: 

 a. create alias script line `configDevContainer`

 b. copy all the base dev container setup in the .devcontainer folder in this repo, feel free to make any customised modification for container setup


# Usage

Go to the repo where u wanted to use devcontainer in vscode and run: 

```bash
configDevContainer
```

This will create a .devcontianer folder in your desinated repo and you can start using dev container for development :D
